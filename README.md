# 🌡 K-Type Thermocouple Calculator

Простой и удобный веб-калькулятор для расчёта температуры по показаниям K-термопары. Подходит для мультиметров, которые измеряют напряжение в милливольтах, но не умеют напрямую работать с термопарами.

## 🎯 Зачем это нужно?

Большинство бюджетных мультиметров измеряют только напряжение (мВ), но не имеют режима термопары. Этот калькулятор помогает быстро перевести показания в температуру с учётом температуры воздуха (компенсация холодного спая).

## 🔬 Формула расчёта
T = T_room + (V_mV / 0.041)

Где:
- **T** — измеряемая температура (°C)
- **T_room** — температура воздуха (°C)
- **V_mV** — напряжение термопары (мВ)
- **0.041** — чувствительность K-термопары (мВ/°C)
Возможности

- Быстрый ввод напряжения (кнопки 0, 0.5, 1, 2, 5 мВ)
- ыстрый ввод температуры воздуха (0°, 20°, 25°, 30°C)
- Отображение разницы температур (ΔT)
- Адаптивный дизайн для мобильных устройств
- Мгновенный пересчёт при изменении любого параметра
- Тёмная тема для комфортной работы

## 🚀 Использование

1. Измерьте напряжение на выходе термопары мультиметром (в режиме мВ)
2. Введите значение в поле "Напряжение термопары"
3. Введите температуру воздуха в помещении
4. Получите точную температуру измеряемого объекта

Пример
- Напряжение: **0.50 мВ**
- Температура воздуха: **25°C**
- Результат: **37.2°C** ✅ (температура тела)

## 🔧 Технические детали

- Чувствительность K-термопары: **41 мкВ/°C** (0.041 мВ/°C)
- Погрешность: **< 1%** (при точном вводе данных)
- Поддержка отрицательных температур: да
